# Diseño de la Página Web - Inducción a la Administración Pública

## Filosofía de Diseño

### Colores Institucionales
- **Azul Primario**: #0f2b5b (azul institucional fuerte)
- **Azul Secundario**: #1e40af (azul institucional)
- **Blanco**: #ffffff (fondo principal)
- **Gris Neutro**: #f8fafc (fondo secundario)
- **Texto Principal**: #1e293b (gris oscuro)
- **Texto Secundario**: #64748b (gris medio)

### Tipografía
- **Encabezados**: Inter (sans-serif moderna)
- **Cuerpo**: Source Sans Pro (legible y profesional)
- **Código**: JetBrains Mono

### Estilo Visual
- **Diseño**: Moderno, institucional, profesional
- **Layout**: Grid responsivo con espaciado generoso
- **Efectos**: Animaciones sutiles y transiciones fluidas
- **Elementos**: Cards modernas, botones con hover effects

## Estructura de Navegación

### Menú Principal
1. **Inicio** - Página principal con introducción
2. **Unidad I** - Nociones sobre Organización del Estado
3. **Unidad II** - Fines del Estado y Administración Pública
4. **Unidad III** - Marco Jurídico de la Administración Pública
5. **Régimen Función Pública** - Ley 41-08
6. **Casos Prácticos** - Ejemplos y ejercicios
7. **Glosario** - Términos y definiciones

## Efectos Visuales

### Animaciones
- **Scroll Reveal**: Elementos aparecen al hacer scroll
- **Hover Effects**: Botones y cards con transformaciones
- **Progress Indicators**: Barras de progreso para las unidades
- **Interactive Elements**: Acordeones, tabs, modales

### Componentes Interactivos
1. **Acordeón de Contenido** - Desplegable por secciones
2. **Tabs de Navegación** - Cambio entre temas
3. **Modal de Casos** - Ventanas emergentes con ejemplos
4. **Progress Tracker** - Seguimiento del progreso del curso
5. **Interactive Quiz** - Preguntas de autoevaluación

### Background y Decoraciones
- **Gradiente Sutil**: Azul a blanco en el header
- **Patrones Geométricos**: Elementos decorativos minimalistas
- **Sombras**: Profundidad en cards y elementos flotantes
- **Iconos**: Set de iconos institucionales consistentes

## Layout Responsivo

### Desktop (1200px+)
- Header con navegación horizontal
- Sidebar con menú de unidades
- Contenido principal en área central
- Footer institucional

### Tablet (768px - 1199px)
- Menú hamburguesa
- Layout de dos columnas
- Cards adaptables

### Mobile (< 768px)
- Menú desplegable
- Layout de una columna
- Tipografía optimizada para móvil

## Características Especiales

### Accesibilidad
- Contraste 4.5:1 mínimo
- Navegación por teclado
- Screen reader friendly
- Textos alternativos en imágenes

### Performance
- Optimización de imágenes
- CSS minificado
- JavaScript asíncrono
- Lazy loading

### SEO
- Meta tags descriptivos
- Estructura semántica HTML
- Open Graph tags
- Schema markup
